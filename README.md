## Общая информация
- Версия java: "25.0.2" 2026-01-20 LTS
- Система сборки: не используется (javac)
- Сторонние библиотеки: не используются

## Инструкция по запуску:
1. Скачать репозиторий с проектом
2. Открыть директорию с проектом в командной строке
3. Ввести команду для сборки проекта: "javac *.java"
4. Ввести команду для запуска проекта: "java SortApp [options] `<input files>` "
## Options:
    -o <path>  output directory (путь к директории вывода)
    -p <pref>  prefix for output files (префикс для файлов вывода)
    -a         append mode (вывод в режиме дополнения)
    -s         short statistics (режим краткой статистик)
    -f         full statistics (режим полной статистики)
  
## Особенности реализации: 
- По умолчанию задается режим с краткой статистикой
- Любая строка с пробелами считается как "string"
- В репозитории есть файлы in1 и in2 с входными данными из примеров указанных в ТЗ
